## About

This project is my solution to the Frontend Mentor "Bento Grid" challenge. It demonstrates a responsive bento-style layout built primarily with modern CSS Grid (with a mobile-first approach) and semantic HTML. The layout adapts across viewports using CSS Grid areas and responsive breakpoints. The repository focuses on clean, accessible markup and pure CSS ‚Äî no JavaScript required.
